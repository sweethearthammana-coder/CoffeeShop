# Coffee Shop App — Hosting Guide

## How to put this on your phone (free, no App Store)

### Step 1 — Create a free GitHub account
1. Go to **github.com**
2. Click **Sign up** — use any email
3. Choose the free plan

### Step 2 — Create a new repository
1. Click the **+** icon (top right) → **New repository**
2. Name it: `coffeeshop`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload the files
1. Click **uploading an existing file** (on the empty repo page)
2. Upload ALL files:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - The `icons` folder (drag both icon files)
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to **Settings** (top tab of your repo)
2. Scroll to **Pages** (left sidebar)
3. Under "Branch" select **main** → click **Save**
4. Wait 1–2 minutes
5. Your app URL will be: `https://YOUR-USERNAME.github.io/coffeeshop`

### Step 5 — Install on iPhone
1. Open the URL in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (square with arrow, bottom of screen)
3. Tap **"Add to Home Screen"**
4. Tap **Add**
5. The app appears on your home screen like a real app!

### Step 5 — Install on Android
1. Open the URL in **Chrome**
2. Tap the **⋮ menu** (top right)
3. Tap **"Add to Home screen"**
4. Tap **Add**
